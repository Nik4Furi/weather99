## <h2 id="runLocally" >Run Locally </h2>

Clone the project

```bash
  git clone https://github/Nik4Furi/weather99
```

Go to the project directory

```bash
  cd weather99
```
Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
